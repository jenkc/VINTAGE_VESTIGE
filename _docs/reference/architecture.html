Phase Roadmap

Each phase builds on the intelligence layer — not just adds features

Phase 1 · Now → Month 3
Build the Graph
  • Backfill platform in Qdrant payloads
  • StyleBridge model + migration
  • compute_bridges.py script
  • Bridge narratives via Claude
  • FastAPI backend (search + bridges)
  • Connect vv-web to API
  • Style Ancestry + Modern Echoes UI
Target: 5K–10K bridge rows · ~$9 total
Phase 2 · Month 3–6
Style DNA API
  • POST /analyze endpoint
  • GET /timeline/:attribute
  • API key auth + rate limiting
  • Usage tracking + Stripe billing
  • First B2B customers
  • 1,000 → 5,000 enriched items
Target: first paying customer · $99–$499/mo tiers
Phase 3 · Month 6–12
Scale the Data Layer
  • V&A Museum API (200K+ items)
  • Europeana Fashion (500K+ items)
  • iDesigner runway images
  • NYPL Digital Collections
  • 5,000 → 50,000+ enriched items
  • Network effect on bridges
Target: $10K ARR · 3+ data sources
Phase 4 · Month 9–15
Trend Oracle
  • Temporal bridge momentum signals
  • Runway collection DNA analysis
  • Resale value prediction
  • Annual subscriptions
  • Competes with WGSN ($30K+/yr)
Target: $5K–50K/yr per customer
